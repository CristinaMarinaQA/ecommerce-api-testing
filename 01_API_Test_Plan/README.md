# API Test Plan – Ecommerce API

## Objective
The objective of this test plan is to validate the functionality, reliability, and correctness of the Ecommerce REST API.

## API Under Test
https://fakestoreapi.com

## Scope of Testing

The following API endpoints are tested:

- GET /products
- GET /products/{id}
- POST /products
- PUT /products/{id}
- DELETE /products/{id}

## Test Types

The following types of testing were performed:

- Functional API Testing
- Positive Testing
- Negative Testing
- Data Validation
- Response Validation

## Tools Used

- Postman
- GitHub

## Test Deliverables

The following artifacts are included in this project:

- API Test Plan
- API Test Scenarios
- Postman Collection
- Test Execution Report
- Bug Reports
